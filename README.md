# Laravel Inertia Strart 2026 by Ecomsys 

## Tech Stack

- Laravel 13.x
- PHP 8.5+
- SQLite / MySQL
- Tailwind CSS 4.x
- Shadcn/ui

## Installation from GitHub repository

```bash
git clone https://github.com/ecomsys/laravel-inertia-start.git
cd laravel-inertia-start
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
composer run dev
```
